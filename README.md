# StudyFlow

StudyFlow will begin as a simple HTML/CSS/JavaScript assignment tracker and gradually evolve into a full-stack web application as I learn React, backend development, databases, authentication, and deployment.

## Planned Tech Stack

### Current Version
- HTML
- CSS
- JavaScript
- Git/GitHub

### Future Full-Stack Version
- Frontend: React
- Backend: Node.js + Express
- Database: PostgreSQL
- Authentication: User login/signup with password hashing
- API Style: REST API
- Deployment:
  - Frontend: Vercel
  - Backend: Render or Fly.io
  - Database: Supabase or Neon

### Future Features
- Add, edit, delete, and complete assignments
- Organize assignments by class
- Filter assignments by due date, priority, and completion status
- User accounts
- Personal dashboard
- Database persistence
- Optional AI-generated study plan feature
